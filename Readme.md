# 📚 BookInn

BookInn is a modern book-themed website designed and developed using pure HTML and CSS. The project focuses on creating an attractive, responsive, and user-friendly interface for book enthusiasts without relying on external frameworks or libraries.

## Overview

The website showcases a clean and visually appealing design that highlights books, categories, and featured content. It demonstrates front-end development skills through structured layouts, responsive design principles, and effective use of CSS styling.

## Features

* Modern and responsive user interface
* Clean and organized layout
* Pure HTML5 and CSS3 implementation
* Mobile-friendly design
* Easy navigation
* Book showcase sections
* Reusable and maintainable code structure

## Technologies Used

* HTML5
* CSS3


## Getting Started

To run the project locally:

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Launch `index.html` in your preferred web browser.
